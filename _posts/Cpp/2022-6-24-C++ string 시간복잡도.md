미리 보는 결론
1. C++ std::string의 +', '+=' 연산자 오버로딩의 시간복잡도는 다르다.
2. str += 'a';	// 시간복잡도 O(1)
3. str = str + 'a'	// 시간복잡도 O(n)
4. str = 'a' + str	// 시간복잡도 O(n)

```cpp
#include <iostream>
#include <string>
using namespace std;

int main()
{
	string str;
	const int n = 1'000;
	for (int i = 0; i < n; i++)
	{
		str += 'a';
	}
	return 0;
}
```
