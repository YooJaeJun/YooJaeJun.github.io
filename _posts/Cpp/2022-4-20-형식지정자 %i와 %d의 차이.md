printf와 같은 출력 형식 지정자에는 동일
<br><br>
scanf와 같은 입력 형식 지정자에는
1. %i
<br>
10진수/8진수/16진수 입력받음
<br>
2. %d
<br>
signed 10진수 정수를 입력받음
<br>

```cpp
#include <stdio.h>

int main() {
    int n1, n2;
    scanf("%i", &n1);
    scanf("%d", &n2);

    printf("%d\n", n1);
    printf("%d", n2);
    return 0;
}
```
```cpp
입력:
011
011
결과:
9
11
```
