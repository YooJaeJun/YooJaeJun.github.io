# const

- 상수를 선언하는 키워드

- const로 선언된 변수, 포인터, 멤버함수, 객체를 상수화 시킨다.

<br>

# const 위치

- int와 같은 빌트인 타입을 상수화할 때 const는 타입 앞뒤 무관하다.

```cpp
#include <iostream>
using namespace std;

int main()
{
	int num = 10;
	
	const int constNum = num;
	int const constNum = num;	// 둘 다 똑같다.
}
```
